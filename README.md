# Themis

A declarative data-center-fabric lab. One Linux host, a short Themisfile, and you have a running routed fabric with realistic BGP, a working control plane, and composable chaos.

Built for enterprise IT: network engineers, SREs, and architects who run BGP and need to rehearse changes before production. Protocol researchers with hypotheses to test against real routing stacks.
